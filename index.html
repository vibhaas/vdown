Vdown

Download files from a file which has a list of downloads(python)

View project on GitHub

V down is a small python program made for downloading a list of downloads mentioned in a in another file. For running this code you will need to install python(Remember install python 2). Run it by typing- `$ python vdown.py` in your command line terminal(type every thing after the dollar symbol). Once started it will display some thing like this-

            V Down             

Enter file name:

Type the file name of your down list file. I have given test.txt for an example.

Down List file

You have to create another file for a list of downloads, I have given the syntax here-

Download: (download url): (download file name)

Example-

Download: http://www.vibhaas.org: vib.html

You can-

  • Change the directory but you have to create the directory with that name first
  • Type text which is not from the syntax
  • Download image files, html files, audio files, video file etc.
